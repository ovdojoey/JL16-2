Welli

Welli is a health tracking website that allows for a simple way to track your life over time. It works by letting you rate things on a scale of 0 to 10. After you set a rating, you can add tags which are stored along with the rating. I built the project using PHP and jQuery.

THE TECH

Welli was built with PHP, mySQL, jQuery, HTML, and Less. It also features Stripe for payment processing.
